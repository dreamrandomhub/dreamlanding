# Landing Page Prototype

Прототип личного лэндинга для портфолио.
Стек: Next.js 15, React 19, Inline Styles (для быстрого прототипирования).

## Запуск

```bash
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000).

## Структура

- `app/page.js` - Основная страница (Single Page)
- `app/globals.css` - Глобальные стили и шрифты
- `public/images` - Ассеты
