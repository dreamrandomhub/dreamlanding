## Язык и стиль коммуникации

- Язык общения - **русский**
- Логи и комментарии в коде - **на русском**

## Старт и общение

- README.md - описание и паспорт проекта
- flow общения: запрос - анализ - план - подтверждение/уточнение плана от пользователя - код

## Планирование изменений

При обсуждении/составлении плана изменений/интеграций:
- Отвечай в стиле **бизнес-логики и flow**, без кусков кода
- Предлагай **расширенные решения (best practices)**, особенно когда пользователь спрашивает совет

### После завершения изменений

1. npm run typecheck
2. npm run lint
3. npm run build
